# ts-starter

An boilerplate for typescript projects composed of small shell scripts.

## Setup

Run

```bash
yarn setup
```

or

```bash
npm run setup
```

## Scripts

- `yarn setup-jest`: add jest dependencies and `jest.config.js`
- `yarn setup-express`: add express app dependencies and a simple express boilerplate
- `yarn reset`: reset project to initial state (before `yarn setup`)
